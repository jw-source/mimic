# Mimic

#### Create a venv
```
python3.11 -m venv venv
```
#### Activate the venv
```
source venv/bin/activate
```
#### Install python packages
```
pip install -r requirements.txt
```
#### Convert .env.txt file to .env (add Gemini & OpenAI API keys)
```
mv .env.txt .env
```
#### Run the run.py file
```
python run.py
```
